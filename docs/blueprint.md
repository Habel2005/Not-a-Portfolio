# **App Name**: VOID ARCHIVE OS

## Core Features:

- Spatial Force Field Navigation: An alternative to traditional scrolling where users navigate an infinite canvas using cursor-driven physics and gravity-based momentum.
- Architectural Kinetic Typography: Massive headlines that scale, mutate, and fracture in 3D space, acting as structural landmarks rather than simple text headers.
- Fluid Displacement Portals: Navigational transitions using GLSL shaders to create liquid chrome and glass distortion effects between data environments.
- Geometric Shard Persistence: A dynamic system using Firestore to store project metadata, which is then rendered as floating, interactive geometric fragments within the OS.
- Magnetic Interaction Matrix: A cursor system that uses attraction and repulsion physics to manipulate interface elements, creating a tactile, living environment.
- Infinite Zoom Hierarchy: A non-linear exploration system that replaces pages with dimensional zoom levels, allowing users to dive into project details through spatial depth.

## Style Guidelines:

- Deep Obsidian (#0D0D0E) and Stealth Black (#050505) base with a high-precision Pulsating Acid Green (#D2FF00) accent.
- Extreme scale contrast using Neue Montreal and Geist for technical rhythm, with Canela for high-fashion editorial highlights. Headlines behave as 3D sculpture.
- Anti-standard, non-grid architecture using physics-simulated asymmetry and layered depth systems. No traditional boxes or navigation bars.
- Weighted cinematic motion (0.8s–1.4s) using GSAP and Lenis, featuring chromatic aberration, volumetric lighting, and displacement blur.
- Abstract, brutalist line symbols and high-fashion graphics that react to proximity fields rather than serving as literal icons.
